# SP-1: Quality-per-Dollar (QpD) Eval Meter — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:executing-plans or subagent-driven-development. Strict TDD — every behavior gets a failing test first. Steps use `- [ ]`.

**Goal:** Build the objective function the whole project optimizes against ("meter before engine"): drive the Misul agent headlessly on deterministic Tier-1 fixtures in isolated dirs, capture real per-run token+cost from provider usage, score by running each fixture's tests, report `QpD = mean(task_score)/mean(cost_usd_per_task)` + `cost_of_pass`, and support matched-pairs A/B (baseline vs variant, ≥3 seeds) with a McNemar significance gate.

**Architecture:** New npm-workspace package `packages/eval` (TS/tsgo/vitest/biome, like SP-0). Drives the in-repo `@earendil-works/pi-coding-agent` via its SDK `createAgentSession()` → `session.prompt()` → `session.getSessionStats()`. Cost truth = pi-ai's pre-computed `Usage.cost.total`, already summed by `getSessionStats()`. Cherry-pick oh-my-pi `run-collector.ts` for per-step granularity + cross-check (3 import swaps + ~10-line adapter); build a ~40-line JSONL aggregator from scratch (NOT the Bun stats parser). Subprocess + temp-dir isolation (no Docker). Offline-by-default tests via the faux provider.

**Stack:** TS (tsgo), npm workspaces, vitest 3.2.4, Node ≥22.19. New dep: `@opentelemetry/api` (type-only, for run-collector). No Bun/Rust/Docker.

## Key grounding facts (verified — do not re-derive)
1. Cost is provider-authoritative + pre-computed: `Usage.cost = {input,output,cacheRead,cacheWrite,total}` (`ai/src/types.ts:274-289`), filled via `calculateCost()` (`ai/src/models.ts:39`). No tokenizer in the cost path.
2. SDK aggregates per session: `getSessionStats()` returns `{ tokens, cost, ... }`.
3. Headless: in-process `createAgentSession({cwd, model?, tools, sessionManager: SessionManager.inMemory()})` → `await session.prompt(text)` → `getSessionStats()`; subscribe via `session.subscribe(e => ...)` for `message_end`.
4. Fixture contract: `<dir>/{prompt.md, input/, expected/, metadata.json}`; Tier-1 grading = run `metadata.testCommand`, exit 0 = pass (binary, zero judge cost).
5. Faux provider: `registerFauxProvider({api,provider,models})` + `.setResponses([...])`, `estimateTokens=ceil(len/4)` → deterministic offline runs, zero API cost.
6. Noise floor: single-run deltas <~15pp = noise; require ≥3 seeds + McNemar gate for A/B.
